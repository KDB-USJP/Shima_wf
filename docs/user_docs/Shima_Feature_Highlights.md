# Shima Tutorial: Feature Highlights 🏝️✨

---

## 🏎️ The OmniJog & Demuxers
*   **One-Stop Control**: Manage up to 20 parameters from a single shuttle wheel.
*   **Accelerated Stepping**: Precision adjustment with self-accelerating hold loops.
*   **Wired Logic (MUX)**: A single cable carries all channel data to downstream nodes.
*   **Smart Labels**: Digital readouts sync with DemuxList labels for human-readable controls.
*   **Color Coding**: Per-channel LED colors for instant visual recognition.

## 🤖 Model Citizen & The BNDL Ecosystem
*   **Unified Loading**: Checkpoint, CLIP, and VAE in one optimized footprint.
*   **LoRA Stacking**: Integrated stack support for complex style management.
*   **The BNDL Standard**: "One-Wire" connectivity—package your entire model suite into a single link.
*   **Panel Mode**: Compact PCB-style chassis for minimal, ultra-clean workflow layouts.
*   **External Linking**: Toggle "Use Everywhere" matching with a single click (The Topbar 'X').

## ✍️ Master Prompt (Conditioning)
*   **Model-Aware Inputs**: Dynamically switches between SDXL, SD1.5, SD3, and Flux modes.
*   **Multi-Encoder Power**: Independent control over CLIP-L, CLIP-G, and T5-XXL text encoders.
*   **Weighted Encoding**: Fine-tune the "influence" of each encoder with precision sliders.
*   **The Prompt BNDL**: Send Positive and Negative conditioning down a single wire.
*   **Legacy-Compatible**: Works with vanilla nodes via the Shima DeBNDLer.

## 🎲 Shima Sampler (Performance Sampling)
*   **The s33d Standard**: Uses external seed control to eliminate ComfyUI's "control_after_generate" widget clutter.
*   **HiRes Fix (Pass 2)**: Integrated latent upscaling with independent denoise and scheduler controls.
*   **Total Connectivity**: Native support for BNDL bundles (Model, Prompt, Latent) and CommonParams.
*   **Real-Time Telemetry**: Digital readout shows the exact seed, model type, and sampling source used.
*   **Panel Native**: A virtual macro node that expands into a full efficiency pipeline on execution.

## 🛡️ Shima NSFW Checker (Safety Hub)
*   **Dual-Engine Logic**: Combines ViT-base classification for global scoring with NudeNet for targeted segmentation.
*   **Active Censorship**: Automatically apply Mosaic, Blur, or Black Bar masks to sensitive regions.
*   **The Rating Bug**: Dynamic "On-Image" badges (G, PG, PG13, R, X) for instant safety identification.
*   **Production Lock**: Output a secondary "Safe Image" (No Bug/No Censor) that blacks out automatically if unsafe.
*   **Safe-Mode Fallback**: Graceful failure handling—blacks out the image if AI dependencies are missing.

## 🎞️ Shima SEG Toolkit (Localized Control)
*   **AutoYoloSEG**: Unified BBOX/SEGM detector that auto-detects model types and routes logic instantly.
*   **Interactive Selector**: Pick segments via a visual frontend gallery instead of guessing index numbers.
*   **SEG Math Logic**: Perform Boolean operations (Add, Subtract, Intersect) on complex region lists.
*   **The SEG Sampler**: A "Mega Node" wrapper that abstracts the complexity of DetailerForEach into a single prompt input.
*   **Impact Pack Ready**: Fully compatible with the Impact Pack ecosystem and standard SEGS protocols.

## 👗 Shima Styler (The Collection)
*   **Visual Discovery**: Browse hundreds of art styles with rich thumbnail previews.
*   **Dynamic Gallery**: Large-scale visual selection for finding "the look" instantly.
*   **Auto-Iteration**: Test dozens of styles sequentially with the Style Iterator.
*   **Favorite Selection**: Keep your "Go-To" aesthetics pinned for quick access.
*   **Excel Integration**: Fully customizable styles via simple spreadsheet editing.

## 💾 Multi-Saver (All-In-One Output)
*   **Deep Processing**: Extract Lineart, Canny, Depth, and Normal maps automatically on save.
*   **Mask Generation**: Auto-generate highlight/shadow masks and color palettes.
*   **One-Click Toggle**: Individually enable/disable complex outputs via icon switches.
*   **Metadata Mastery**: Embed custom notes and project tracking into every image.
*   **Zero-Overhead Formatting**: Native support for PNG, JPG, and WebP with quality control.

## 🛠️ Shima Transformer (Atomic Logic)
*   **Atomic Switch Pattern**: Use simple `key ||| value` syntax for complex conditional mapping.
*   **Variable Injection**: Inject dynamic tokens (`!!!`, `@@@`, `###`, `$$$`) into any output string.
*   **Unconditional Overrides**: Instantly pivot any input to a specific value with a single line catchall.
*   **Smart Type Inference**: Automatically converts "true/false" or numbers from text back into native Python types.

## 🌋 The Custodian (System Hub)
*   **Global Toggles**: Hide/Show all Dymo Labels or Headlines with a single button.
*   **VRAM Cleanup**: Recursively nukes cache folders to keep your system lean.
*   **Security & Safety**: Integrated NSFW checker and "Canary" hardware monitors.
*   **Action Feedback**: Real-time digital status readout of all cleanup tasks.
*   **The Gateway**: Instant portals to Help, Documentation, and Style Dashboards.

## 🩺 System Guardians (Maintenance)
*   **Dependency Generator**: Scans your entire workflow to create a "Missing Asset" manifest for sharing.
*   **Workflow Checker**: Real-time diagnostic scan identifies missing models and sub-node errors in the UI.
*   **Workflow Image Creator**: Save pixel-perfect PNGs with embedded workflow metadata for the Shima Gallery.
*   **Package Installer**: Integrated support for deploying missing models directly from the Shima Hub.
